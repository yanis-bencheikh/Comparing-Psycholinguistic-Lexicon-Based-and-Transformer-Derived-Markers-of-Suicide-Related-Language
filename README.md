# Comparing-Psycholinguistic-Lexicon-Based-and-Transformer-Derived-Markers-of-Suicide-Related-Language
Responsible-AI audit comparing psycholinguistic, lexicon-based (n-grams, pronoun density, absolutist words, NRC emotions), and transformer-derived (RoBERTa-GoEmotions) markers of suicide-related language on 232k balanced Reddit posts. Effect sizes via Cohen's d. Not a clinical tool.
